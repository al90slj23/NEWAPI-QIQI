# 页面清单表规范

> 详细规范：[standards/02.backend-04.db-manifest.md](../standards/02.backend-04.db-manifest.md)

## 表命名

```
{系统前缀}_manifest_list
```

举例：`yys_manifest_list`、`admin_manifest_list`

## 核心字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `path` | VARCHAR(500) | 文件夹路径（唯一主体） |
| `url` | VARCHAR(200) | 访问URL |
| `route` | VARCHAR(100) | 路由名称 |
| `shortName` | VARCHAR(50) | 菜单显示名称 |
| `icon` | VARCHAR(50) | 图标 |
| `sortOrder` | INT | 排序顺序 |
| `status` | VARCHAR(20) | 状态 |

## 路径映射

```
path: /resources/js/views/admin/users/list/
        ↓
url: /admin/users/list
route: admin_users_list
```

## Fallback 策略

| 字段 | Fallback 值 | 说明 |
|------|-------------|------|
| `shortName` | 文件夹名 | 快速暴露未配置 |
| `icon` | `help-circle` | 问号图标 |
| `sortOrder` | `-1` | 排最前面 |

## 状态值

| 状态 | 说明 |
|------|------|
| `active` | 活跃 |
| `inactive` | 停用 |
| `development` | 开发中 |

## 同步命令

```bash
php artisan manifest:sync      # 同步文件系统到数据库
php artisan manifest:validate  # 验证一致性
```

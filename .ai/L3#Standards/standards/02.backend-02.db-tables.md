# 数据库表命名规范

> 详细规范：[standards/02.backend-02.db-tables.md](../standards/02.backend-02.db-tables.md)

## 表命名格式

```
{业务域}_{子域}_{通用语义后缀}
```

## 通用语义后缀（单数形式）

| 后缀 | 含义 | 举例 |
|------|------|------|
| `list` | 主表 | `users_list` |
| `config` | 配置表 | `users_config_roles` |
| `log` | 日志表 | `users_log_login` |
| `data` | 扩展数据 | `users_vip_data` |
| `history` | 历史快照 | `orders_log_history` |

## 命名规则

```
通用语义后缀 = 单数（list/config/log/data）
普通业务词汇 = 复数（users/orders/levels）
```

## 举例

```sql
-- ✅ 正确
users_list
users_config_roles
users_log_login

-- ❌ 错误（后缀用了复数）
users_lists
users_configs_roles
```

## 表注释格式

```sql
'中文名称|功能说明'
-- 举例：'用户主表|存储用户基础信息'
```

## 路径映射

```
文件夹路径：pages/admin/users/config/levels/
数据库表：admin_users_config_levels
```

# API设计规范

> 详细规范：[standards/02.backend-01.api.md](../standards/02.backend-01.api.md)

## 核心规则

**斜杠 `/` 仅用于真实目录，虚拟参数用冒号 `:`**

## 语法对照

| 符号 | 用途 | 举例 |
|------|------|------|
| `/` | 真实目录层级 | `/admin/users/list` |
| `:` | 虚拟参数（ID/操作） | `/list:123`、`/list:export` |
| `::` | 子资源层级 | `/list:123::orders` |

## 举例

```bash
# 基础CRUD
GET    /api/admin/users/list              # 列表
POST   /api/admin/users/list              # 创建
PUT    /api/admin/users/list:123          # 更新 ✅
DELETE /api/admin/users/list:456          # 删除 ✅

# 自定义操作
GET    /api/admin/users/list:export       # 导出 ✅
POST   /api/admin/users/list:batchDelete  # 批量删除 ✅

# 子资源
GET    /api/admin/users/list:123::orders  # 用户的订单 ✅

# ❌ 禁用旧风格
PUT    /api/admin/users/list/123          # 与文件路径混淆
```

## 操作名命名

```
✅ 正确：camelCase
:export、:batchDelete、:refreshCache

❌ 错误
:batch_delete、:REFRESH_CACHE
```

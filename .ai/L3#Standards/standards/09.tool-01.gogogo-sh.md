# 09.tool-01.gogogo-sh - gogogo.sh 统一入口脚本规范

> **当前规范**：`.ai/L3#Standards/standards/09.tool-01.gogogo-sh.md`

---

## 核心理念

**统一入口**：一个命令搞定所有常用操作，数字选择即可。

---

## 文件结构

```
project/
├── gogogo.sh          # 主入口
├── gogogo.lib.sh      # 通用库
├── gogogo.1.sh        # 选项 1: 本地开发
├── gogogo.2.sh        # 选项 2: 部署
├── gogogo.3.sh        # 选项 3: 清缓存
└── ...            # 按需扩展
```

---

## 选项编号

| 编号 | 用途 |
|------|------|
| 0 | 退出 |
| 1 | 本地开发（默认） |
| 2 | 部署 |
| 3 | 清缓存 |
| 4 | 测试 |
| 5 | 构建 |

---

## 输出函数

```bash
success "操作成功"     # ✅ 绿色
error "操作失败"       # ❌ 红色
warn "警告信息"        # ⚠️  黄色
info "提示信息"        # ℹ️  蓝色
step "执行步骤"        # 📌 青色
```

---

## 使用方式

```bash
./gogogo.sh        # 交互式菜单
./gogogo.sh 1      # 直接执行选项 1
./gogogo.sh 2      # 直接执行选项 2
```

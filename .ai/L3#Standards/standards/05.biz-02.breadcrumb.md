# 05.biz-02.breadcrumb - 管理后台面包屑规范

> **文档版本**：v1.0.0
> **创建日期**：2026-01-01

---

## 📍 文档定位

定义管理后台 C 区（页面头部）的面包屑显示格式和交互规范。

---

## 🎯 布局结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ 第一行: 页面:{页面名称}  路径:{完整路径}                [复制] │ [刷新] │
│ 第二行: {页面简介}                                              │ [全屏] │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 第一行（主要信息）

| 元素 | 说明 | 示例 |
|------|------|------|
| 页面标签 | 固定文字 `页面:` | 页面: |
| 页面名称 | 当前页面的 `shortName` 或 `name` | 用户列表 |
| 路径标签 | 固定文字 `路径:` | 路径: |
| 完整路径 | 面包屑路径，用 `/` 分隔 | 调度室/配置中心/站点信息 |
| 复制按钮 | 复制页面信息和 URL | [复制] |

### 第二行（次要信息）

| 元素 | 说明 | 示例 |
|------|------|------|
| 页面简介 | 当前页面的 `shortInfo` 或 `description` | 管理站点基础信息和品牌配置 |
| 默认值 | 无简介时显示 | 暂无页面简介 |

### 右侧操作按钮

| 按钮 | 功能 | 图标 |
|------|------|------|
| 刷新 | 刷新当前页面 | `refresh-cw` |
| 全屏 | 切换全屏模式（可选） | `maximize` |

---

## 📋 数据来源

面包屑数据从 `manifest` 表获取，每一级路径查找对应配置：

| 字段 | 用途 |
|------|------|
| `manifest_display_short_name` | 显示名称（优先） |
| `manifest_display_name` | 显示名称（备选） |
| `manifest_display_icon` | 图标 |
| `manifest_info_short` | 页面简介 |

---

## 📝 复制功能

点击复制按钮，复制以下格式内容：

```
页面:{页面名称}      路径:{完整路径}
{当前页面URL}
```

### 示例

```
页面:站点信息      路径:调度室/配置中心/站点信息
https://m.srl/cmd/cfghub/msrlinfo
```

## 🎨 样式规范

### 颜色

| 元素 | 颜色 |
|------|------|
| 标签文字 | `rgba(255, 255, 255, 0.4)` |
| 页面名称 | `#fff` + `font-weight: 500` |
| 路径文字 | `rgba(255, 255, 255, 0.6)` |
| 简介文字 | `rgba(255, 255, 255, 0.5)` |
| 分隔线 | `rgba(255, 255, 255, 0.15)` |

### 尺寸

| 元素 | 尺寸 |
|------|------|
| C 区高度 | 双行时 `64px`，单行时 `48px` |
| 字体大小 | 第一行 `13px`，第二行 `12px` |
| 按钮尺寸 | `32px × 32px` |

## 💻 实现示例

### HTML 结构

```html
<div class="cmd-page-header">
    <div class="cmd-page-info">
        <div class="cmd-page-info-main">
            <span class="label">页面:</span>
            <span class="page-name">站点信息</span>
            <span class="separator"></span>
            <span class="label">路径:</span>
            <span class="path-value">调度室/配置中心/站点信息</span>
            <button class="cmd-copy-btn" onclick="copyPageInfo()" title="复制">
                <i data-lucide="copy"></i>
            </button>
        </div>
        <div class="cmd-page-info-sub">
            管理站点基础信息和品牌配置
        </div>
    </div>
    <div class="cmd-page-actions">
        <button class="cmd-page-action-btn" onclick="location.reload()" title="刷新">
            <i data-lucide="refresh-cw"></i>
        </button>
    </div>
</div>
```

### JavaScript 复制函数

```javascript
async function copyPageInfo() {
    const pageName = document.querySelector('.page-name')?.textContent || '';
    const pathText = document.querySelector('.path-value')?.textContent || '';
    const currentUrl = window.location.href;

    const textToCopy = `页面:${pageName}      路径:${pathText}\n${currentUrl}`;

    await navigator.clipboard.writeText(textToCopy);
    showToast('已复制页面信息', 'success');
}
```

## 🎯 核心理念

### 管理后台的本质

管理后台的核心需求是：**完整参数的查看和修改**。

对于管理员来说，直接操作数据库信息往往是最直接、最便捷的方式。通用表格组件正是基于这一理念设计的。

### 快速起项目

```
第一阶段：DbTable 快速搭建
├── 数据库表设计（含字段注释）
├── 通用表格组件渲染
├── 基础 CRUD 操作
└── 即可投入使用

第二阶段：按需定制（后期）
├── 自定义显示格式
├── 操作按钮扩展
├── 操作日志记录
└── 权限细分控制
```

## 🏗️ DbTable 组件设计

### 设计原则

| 原则 | 说明 |
|------|------|
| **数据库即文档** | 字段注释作为列标题，无需额外配置 |
| **零配置启动** | 只需指定表名，自动渲染完整表格 |
| **渐进式增强** | 基础功能开箱即用，高级功能按需添加 |
| **管理员友好** | 直接展示数据库真实数据，所见即所得 |

### 最小化实现

**前端 content.html**：
```html
<!-- D区域内容 -->
<div id="dbtable"></div>
<script>
    new DbTable({
        container: '#dbtable',
        apiUrl: '/msrl/Cmd/YourHub/YourPage/yourpage.cmd.php',
        table: 'your_table_name'
    });
</script>
```

**后端 yourpage.cmd.php**：
```php
<?php
require_once __DIR__ . '/../../../db.inc.php';
require_once __DIR__ . '/../../cmd.dbtable.inc.php';

header('Content-Type: application/json; charset=utf-8');

$input = json_decode(file_get_contents('php://input'), true);
$action = $input['action'] ?? '';

$handler = new DbTableHandler([
    'table' => 'your_table_name',
    'primaryKey' => 'id',
    'defaultOrder' => 'id ASC'
]);

switch ($action) {
    case 'list':
        echo json_encode($handler->list($input));
        break;
    case 'create':
        echo json_encode($handler->create($input));
        break;
    case 'update':
        echo json_encode($handler->update($input));
        break;
    case 'delete':
        echo json_encode($handler->delete($input));
        break;
    default:
        echo json_encode(['code' => 400, 'msg' => '未知操作']);
}
```

### ⚠️ 常见错误

| 错误 | 原因 | 正确写法 |
|------|------|----------|
| `DbTable.init is not a function` | 使用了静态方法调用 | `new DbTable({...})` |
| `DbTable is not defined` | JS 文件未加载 | 确保 index.html 引入 cmd.dbtable.js |
| `Class "DbTableHandler" not found` | PHP 文件路径错误 | 检查 require_once 相对路径 |
| `网络错误` | API 返回非 JSON | 检查 PHP 错误日志 |

### 关键配置参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `container` | string | ✅ | CSS 选择器，如 `#dbtable` |
| `apiUrl` | string | ✅ | API 地址，绝对路径 |
| `table` | string | ✅ | 数据库表名 |
| `primaryKey` | string | ❌ | 主键字段，默认 `id` |
| `pageSize` | number | ❌ | 每页条数，默认 20 |
| `showSearch` | boolean | ❌ | 显示搜索框，默认 true |
| `showAdd` | boolean | ❌ | 显示新增按钮，默认 true |

### 字段注释驱动

数据库字段注释格式：`序号|分组|列名|说明|是否显示`

```sql
CREATE TABLE example_list (
    id INT AUTO_INCREMENT PRIMARY KEY
        COMMENT '1|核心|ID|自增主键|0',
    name VARCHAR(100) NOT NULL
        COMMENT '2|核心|名称|项目名称|1',
    status TINYINT DEFAULT 1
        COMMENT '3|状态|状态|启用状态|1',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
        COMMENT '4|系统|创建时间|记录创建时间|0'
);
```

组件自动：
- 读取字段注释作为列标题
- 根据"是否显示"控制列可见性
- 根据"分组"组织表单布局

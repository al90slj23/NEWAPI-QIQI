## 1. 功能说明

字段设置对话框提供对数据库字段的完整管理：

- **显示控制** - 控制字段在表格中是否显示
- **排序调整** - 拖拽调整字段显示顺序
- **注释编辑** - 直接编辑字段的序号、分组、名称、说明

## 3. 可编辑字段

| 列 | 说明 | 编辑方式 |
|----|------|---------|
| ≡ | 拖拽手柄 | 拖拽排序 |
| 序号 | 注释第1段 | 输入框 |
| 显示 | 注释第5段 | 复选框 |
| 分组 | 注释第2段 | 输入框 |
| 名称 | 注释第3段 | 输入框 |
| 说明 | 注释第4段 | 输入框 |
| 字段名 | 数据库字段名 | 只读 |

## 4. 字段数据结构

```typescript
interface FieldSettingItem {
  /** 字段名（数据库字段名，只读） */
  name: string;
  /** 序号（注释第1段） */
  order: number;
  /** 分组名（注释第2段） */
  groupName: string;
  /** 中文名（注释第3段） */
  label: string;
  /** 说明（注释第4段） */
  helpText: string;
  /** 是否显示（注释第5段） */
  defaultVisible: boolean;
  /** 是否主键 */
  isPrimary: boolean;
  /** 是否系统字段 */
  isSystem: boolean;
}
```

## 5. 保存变更流程

1. **变更检测** - 对比原始值和当前值，生成变更列表
2. **预览确认** - 显示变更预览对话框，列出所有变更项
3. **更新注释** - 调用后端 API 更新数据库字段注释
4. **刷新缓存** - 清除表结构缓存，重新加载

## 6. 后端 API

**更新字段注释**：

```
POST /api/v1/admin/DatabaseTable.ColumnSettings:updateComments
Content-Type: application/json

{
  "table": "users",
  "columns": [
    {
      "name": "name",
      "comment": "1|基础|用户姓名|用户的显示名称|1"
    },
    {
      "name": "email",
      "comment": "3|基础|邮箱|用于登录和通知|1"
    }
  ]
}
```

**后端实现**：

```php
public function updateComments(Request $request): JsonResponse
{
    $table = $request->input('table');
    $columns = $request->input('columns', []);

    foreach ($columns as $col) {
        $sql = "ALTER TABLE `{$table}` MODIFY COLUMN `{$col['name']}` ... COMMENT '{$col['comment']}'";
        DB::statement($sql);
    }

    return response()->json(['success' => true]);
}
```

## 7. 拖拽排序规则

- **同表内拖拽** - 只能在同一个表的字段之间拖拽
- **跨表禁止** - 不能将主表字段拖到扩展表，反之亦然
- **系统字段锁定** - id、created_at、updated_at、deleted_at 不可拖拽

## 8. 列宽设置功能

### 8.1 功能说明

- 控制列的显示/隐藏
- 调整列的显示顺序（拖拽排序）
- 设置持久化到数据库

### 8.2 数据存储

列设置存储在 `admin_column_settings` 表：

```sql
CREATE TABLE `admin_column_settings` (
  `id` BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
  `table_name` VARCHAR(100) NOT NULL COMMENT '表名',
  `column_name` VARCHAR(100) NOT NULL COMMENT '列名',
  `display_order` INT DEFAULT 0 COMMENT '显示顺序',
  `visible` TINYINT(1) DEFAULT 1 COMMENT '是否显示',
  `width` INT DEFAULT NULL COMMENT '列宽',
  `created_at` TIMESTAMP NULL,
  `updated_at` TIMESTAMP NULL,
  UNIQUE KEY `table_column` (`table_name`, `column_name`)
) COMMENT='列设置|存储用户自定义的列配置';
```
